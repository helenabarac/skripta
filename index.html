HTML Styles - CSS

Što je CSS?

  • - Cascading Style Sheets (CSS) se koristi za formatiranje izgleda web stranice.
  • - Pomoću CSS-a možete kontrolirati boju, font, veličinu teksta, razmak između elemenata, kako su elementi pozicionirani i raspoređeni, koje će se pozadinske slike ili boje pozadine koristiti, različite prikaze za različite uređaje i veličine zaslona i još mnogo toga!

CSS se može dodati u HTML dokumente na 3 načina:

  • Inline - korištenjem style atributa unutar HTML elemenata
  • Internal - korištenjem style elementa u head odjeljku
  • External - korištenjem link elementa za povezivanje s vanjskom CSS datotekom

CSS boje, fontovi i veličine

  • - CSS color svojstvo definira boju teksta koja će se koristiti.
  • - CSS font-family svojstvo definira font koji će se koristiti.
  • - CSS font-size svojstvo definira veličinu teksta koja će se koristiti.

CSS obrub

  • - CSS border svojstvo definira okvir oko HTML elementa.

CSS ispuna

  • - CSS padding svojstvo definira padding (razmak) između teksta i obruba.

CSS margina

  • - CSS margin svojstvo definira marginu (razmak) izvan obruba.

Veza na vanjski CSS

  • - Vanjski stilski listovi mogu se referencirati s punim URL-om ili s putanjom relativnom u odnosu na trenutnu web stranicu.
  • Ovaj primjer koristi puni URL za povezivanje sa stilskim listom: >link rel="stylesheet" href="https://www.w3schools.com/html/styles.css"<
  • Ovaj primjer povezuje na stilski list koji se nalazi u html mapi na trenutnoj web stranici: >link rel="stylesheet" href="/html/styles.css"<
  • This example links to a style sheet located in the same folder as the current page: >link rel="stylesheet" href="styles.css"<
